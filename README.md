# Personal Portfolio

This repository contains the source code for my portfolio website, showcasing my skills, projects, and more. The website is built using React.js and includes a variety of features to enhance user experience.

## Features

- **Landing Page**: The introductory section with a brief overview.
- **Skills Section**: Highlights my technical skills and proficiencies.
- **Projects Section**: Displays my recent projects with descriptions and links.
- **About Me Section**: Provides information about my background and experiences.
- **Contact Section**: Features a contact form that saves data to my personal database.
- **Footer Section**: Includes additional links and information.

## Technologies Used

- **Frontend**: React.js
- **Form Handling**: Formik
- **Form Validation**: Yup
- **Dark/Light Mode Toggle**: Custom toggle switch
- **Styling**: Tailwind CSS

## Installation

1. Clone the repository:

   ```bash
  https://github.com/ronit-ghosh/Portfolio.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Portfolio
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```

   Your portfolio website should now be running on [http://localhost:5173](http://localhost:3000).
